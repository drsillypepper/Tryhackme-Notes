1. Your computer checks if local cache has the ip address of the looked up address.
2. if no ip was found, then the request is send to recursive dns server. (usually provided by your isp). Recrusive dns server has the cache of recently looked up domain names. If there request cannot be found from the cache, the request is sent to the root server.
3. The root server act as backbone of the internet. It resolves that in which tld dns server the given domain could recide and forwards the request to the given authoritative server.
4. The tld server server holds records of for where to find the authoritative server to answer the dns request.
5. The authoritative dns server is the server thta is responsible for storing the dns records for a particular domain name etc.
