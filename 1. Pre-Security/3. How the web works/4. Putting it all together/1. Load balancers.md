Load balancers mainly balances the load between multiple instances of applications that are run at bigger scale. Load balancers provide two main features, ensuring high traffic websites can handle the load and providing a failover if a server becomes unresponsive.
- round-robin is common balancing technique where the load is being rotated evenly between receiving servers.
- health-check is another way. If the server doesnt repsond appropriately the putting load will halt until the server responds appropriately again.
