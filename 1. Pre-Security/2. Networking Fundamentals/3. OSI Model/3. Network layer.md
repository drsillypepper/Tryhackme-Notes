In this layer happens routing, determining which paths these chunks of data should be send to be the most optimal.
- These protocols include 
	- **OSPF** (**O**pen **S**hortest **P**ath **F**irst) 
	- **RIP** (**R**outing **I**nformation **P**rotocol)

There are many deciding factors that define what route the packets take
- which path is the shortest(least amount of devices needed)
- what path is the most reliable(depends on many factors, load, etc)
- which is the fastes in overall data transmission capacity

Everything in this layer is dealt with ip addresses. Devices such as routers are capable delivering packets using ip addresses(layer 3 devices they are).
