Layer 4 handles the data accross endpoints in a mannerism based on the protocols being used in this layer.

those are usually either
- tcp (transmission control protocol)
- udp (user datagram protocol)

## TCP
- protocol that quarantees the accuracy of data by making sure all packets arrives without errors by checking packets checksums and makes sure that all packets get delivered to the destination on the network
- capable of synchronizing two devices preventing data flood
- slower than udp due to need to check and synchronize all data between devices.
- used in things such as file sharing, internet browsing, and email for example.
## UDP
- protocol that just sends all packets without checks for whether they got into the right destination and whether the end device received the data properly and uncorrupted.
- the user software is left to decide how fast the packets are being send over the network(the network itself is incapable of deciding that)
- does not reserve continuous connection to the end device
- used in video streaming, games etc where connection speed is favored.
