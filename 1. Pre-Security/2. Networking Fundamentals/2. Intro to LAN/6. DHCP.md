Ip addressed can be assigned by the DHCP server.(or manually)
- dhcp stands for **dynamic host configuration protocol**


## how dhcp works?
devices that are yet to have a ip address on a given network, send a (**dhcp discover**) packet that asks for ip address. The dhcp server then replies with an ip address the device coudl use(**dhcp offer**). The device then request whether it can use the offered ip address(**dhcp request**). The server replies with aknowledgement message(**dhcp ack**).
